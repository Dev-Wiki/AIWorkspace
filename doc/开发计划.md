# AI Workspace 开发计划

> 最后更新：2026-05-19

---

## 项目初心

统一各种 AI Agent 工具（Cursor / Claude / Gemini / Windsurf / Codex / …）的 skill、rule、mcp 配置，以 `~/.ai/` 为全局唯一源，通过 junction/symlink 同步到各 Agent 配置目录。

```
One Workspace → Multiple AI Agents → Unified Capability Management
```

## 三、V1.0 开发任务（目标：全局能力管理完整可用）

### P0 — 必须完成（核心闭环）

#### 1. mcps 完整功能

**现状**：目录已创建（`~/.ai/mcps/enabled` + `disabled`），但无 Agent 目标、无同步逻辑。

**原因**：MCP 配置在各 Agent 中是单个 JSON 文件（非目录），不能直接 junction。需要 Adapter 格式转换。

**实现方案**：

- 后端：在 `~/.ai/mcps/enabled/` 中管理 MCP 定义文件（统一 JSON 格式），同步时按目标 Agent 格式合并生成
- 新增 Agent 目标：
  - Cursor MCP → `~/.cursor/mcp.json`
  - Claude Desktop MCP → `%APPDATA%/Claude/claude_desktop_config.json`
  - Windsurf MCP → `~/.windsurf/mcp.json`
- 同步逻辑：读取 enabled 目录 → 按 Agent 格式生成/合并配置文件 → 备份已有 → 写入
- UI：mcps tab 可用，支持增删改条目

**预估**：300-400 行 Rust + 100 行前端

#### 3. mcps 增删功能

**现状**：skills/rules/plugins 只能启用/禁用已有目录条目，不能新建/删除。

**需要**：
- 后端：`create_item` / `delete_item` 命令
- 前端：列表项旁加 "+" 新建 / "✕" 删除按钮
- 对 skills/rules/plugins 是同逻辑，但 mcps 最需要（常新增 MCP 服务器）

**预估**：80 行 Rust + 80 行前端

---

### P1 — 应该完成（体验完善）

#### 4. 搜索跨类型支持

**现状**：搜索仅在当前 tab 内过滤。

**需要**：全局搜索框，跨 skills/rules/plugins/mcps 搜索，点击结果自动切换 tab 并定位。

**预估**：60 行前端

---

#### 5. 批量操作增强

**现状**：全选/反选仅对当前搜索可见条目。

**需要**：
- "全选全部"（无视搜索过滤）
- 批量移动到指定 Agent
- 操作确认 + 撤销提示

**预估**：100 行前端

---

#### 6. 同步状态可视化

**现状**：日志面板纯文本，需手动查看。

**需要**：
- 每个 Agent 旁显示同步状态图标（已链接 / 未链接 / 冲突 / 错误）
- 同步完成后 toast 提示而非仅有日志

**预估**：80 行前端

### P2 — 可以延后（锦上添花）

**预估**：独立 Rust binary + 跨平台发布，约 500 行

---

#### 8. 冲突检测与合并

**现状**：目标已有非托管内容时直接备份。

**需要**：检测到冲突时提示用户选择（覆盖 / 合并 / 跳过），合并时保留双方内容。

**预估**：150 行 Rust

---

#### 9. 多仓库支持

**现状**：单一仓库路径。

**需要**：支持多个仓库源（如团队仓库 + 个人仓库），按优先级合并。

**预估**：架构改动，200+ 行 Rust + 100 行前端

## 四、V2.0 开发任务（目标：项目级 Workspace + Adapter 架构）

### 核心架构变更

从 **全局 `~/.ai/`** 扩展为支持 **项目级 `.ai/`**：

```
~/.ai/                        # 全局能力库（V1 已有）
  skills/enabled/
  rules/enabled/
  mcps/enabled/

project/.ai/                 # 项目级工作区（V2 新增）
  workspace.json
  profiles/
  agents/
  adapters/
  generated/
```

### V2 任务列表

| # | 模块 | 内容 | 量级 |
|---|---|---|---|
| 1 | 项目初始化 | `aiws init` 创建 `.ai/`，`workspace.json` | 小 |
| 2 | profiles/ | 能力组合配置（类似 VS Code Profile），引用 prompts/rules/mcp | 中 |
| 3 | agents/ | Agent 行为定义（JSON），引用 prompts/rules/workflows/profiles | 中 |
| 4 | prompts/ | Prompt Library（system/task/reusable/chain），Markdown 编辑管理 | 中 |
| 5 | workflows/ | 多步骤 Workflow 定义（JSON），UI 步骤编排 | 中 |
| 6 | context/ | 项目上下文文档（技术栈/架构/API），Markdown 编辑 | 小 |
| 7 | Adapter 系统 | 从 `.ai/` 自动生成 Cursor/Claude/Continue 等原生格式配置 | 大 |
| 8 | generated/ | 生成配置缓存目录 | 小 |
| 9 | 多项目管理 | 项目列表、切换、批量同步 | 中 |
